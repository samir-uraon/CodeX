Introdution

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the foundational technologies for creating and designing web pages.

HTML consists of a series of elements. Each element represents a part of the page (like headings, paragraphs, images, links, etc.). These elements are like building blocks. You stack them together to create the final web page.

CSS describes how HTML elements should be displayed. It’s like the fashion designer for your web pages. You can change fonts, colors, spacing, column layouts, and even add animations—all with CSS.

1. The Birth of HTML:

(a) In 1980, while working as a contractor at CERN (the European Organization for Nuclear Research), Tim Berners-Lee proposed and prototyped a system called ENQUIRE. This system allowed CERN researchers to share and use documents.

(b) In 1989, Berners-Lee took things a step further. He wrote a memo proposing an Internet-based hypertext system—a system that would allow users to navigate interconnected documents using hyperlinks.

(c) As part of this vision, he specified HTML (HyperText Markup Language), which would define the content and structure of web pages.

2. The Birth of the World Wide Web:

(a) In late 1990, Tim Berners-Lee not only specified HTML but also wrote the browser and server software necessary to make it all work.

(b) The following year, in 1991, he made the first-ever website available to the public. It was a simple page that explained what the World Wide Web was and how to use it.

(c) This marked the birth of the World Wide Web as we know it today.

3. The Birth CSS:

On October 10, 1994, Lie submitted his proposal for CSS, outlining how it could solve the problem of document presentation being solely controlled by HTML.

His idea gained traction, and CSS became an essential part of the web development toolkit.

History

HTML History:

  • 1989: Tim Berners-Lee invented the World Wide Web.
  • 1991: Tim Berners-Lee created HTML.
  • 1995: HTML 2.0 was defined.
  • 1999: HTML 4.01 became a W3C recommendation.
  • 2000: XHTML 1.0 was recommended.
  • 2008: WHATWG introduced HTML5.
  • 2014: HTML5 became a W3C recommendation.
We're currently following the latest HTML5 standard.

CSS History:

  • Early Beginnings (1994-1995): Hakon Wium Lie proposed CSS while working at CERN with Tim Berners-Lee, aiming to separate content from layout.
  • CSS1 (1995): The first CSS specification (CSS1) was published by the World Wide Web Consortium (W3C). CSS1 introduced basic styling capabilities like fonts, colors, and margins.
  • CSS2 (1998): CSS2 was released, introducing new features such as positioning, z-index, and media types, which allowed for more sophisticated and adaptable layouts. It also included support for tables and the :hover pseudo-class.
  • CSS2.1 (2011): CSS2.1 became a W3C Recommendation, refining and improving CSS2 by addressing ambiguities and inconsistencies. It became widely adopted and served as a foundation for future developments.
  • CSS3 (2000s-2011): CSS3 began development as modular specifications. It introduced significant new features, including advanced selectors, media queries, and new layout modules (like Flexbox and Grid). CSS3 was released in multiple modules, each evolving independently.
  • Modern CSS (2010s-present): Ongoing advancements include the introduction of new features like custom properties (variables), advanced grid layouts, and new CSS functions. The specification is now developed in an iterative and modular fashion, with improvements and new features being added regularly.
